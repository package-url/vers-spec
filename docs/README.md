This README explains the organization of documentation files for the VERS
specification.

## VERS specification documentation
The file stored in `purl-spec/docs/standard/` are expected to be used
in the VERS standard which we plan to submit to Ecma International as
a new standard in 2026.

The other files in the `purl-spec/docs` folder are also specification 
documentation, but we are not planning to include them in the future Ecma 
VERS standard. Most of these documents provide information to support 
implementation of the VERS Specification in software tools or databases.

**NB:** Many files in the `vers-spec/docs` folder contain "front matter" that
is used for publishing the content on the www.package.url website.
